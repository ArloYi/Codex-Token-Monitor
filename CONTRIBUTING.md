# 参与贡献

感谢你改进 Codex 额度浮窗。

提交贡献即表示你有权提交相关内容，并同意该贡献按本仓库的 [MIT License](LICENSE) 发布。

## 本地开发

要求 macOS 13 或更高版本，并已安装 Xcode Command Line Tools。

```bash
zsh ./scripts/test.sh
```

该命令会构建应用、运行逻辑与焦点安全检查、校验 `Info.plist` 和代码签名。

## 提交约定

1. 从 `main` 创建功能分支。
2. 一次提交只解决一个明确问题。
3. UI 或窗口行为变更必须说明是否影响焦点、点击穿透和其他应用。
4. 数据口径变更必须附带输入、预期输出和回归测试。
5. 提交前运行 `zsh ./scripts/test.sh`。

请勿提交你无权再分发的代码、素材、商标或数据，也不要提交 Token、Cookie、账号信息、数据库、日志、提示词或回复内容。

## Pull Request

PR 描述应包含变更动机、验证结果、截图或终端输出，以及仍未覆盖的边界。不要提交 `build/`、`dist/`、个人 Codex 状态、数据库或日志。
