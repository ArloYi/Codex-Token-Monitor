## 直接使用

从 GitHub Releases 下载 `Codex-Quota-HUD-*-macos-universal.zip`，解压后双击：

`Codex Quota HUD.app`

如果你是从源码构建，应用会生成在 `build/Codex Quota HUD.app`。

浮窗只在 Codex 位于前台时出现。默认在右上方显示深色额度仪表卡：左侧为环形剩余比例，右侧显示“额度剩余”、本周期已用比例和进度条。鼠标移入后会在原位置下方向下展开项目本周用量与历史总量，移出后自动收起。按住卡片任意位置即可拖动；展开后拖动右下角把手可等比缩放整体。拖到屏幕任一边缘松手，会自动吸附成圆形额度状态球；将状态球拖离边缘则恢复完整卡片。

首次运行时，macOS 可能打开“系统设置 → 隐私与安全性 → 辅助功能”。请打开 **Codex Quota HUD** 开关，然后重新启动浮窗。该权限只用于读取 Codex 顶部当前可见的任务名称，不记录键盘输入，也不会点击 Codex 控件。

## 这次修复与调整

- 修复 Codex 的 `selected-project` 为 `null` 时应用崩溃的问题。
- 修复 rollout Token 事件中的 `info` 为 `null` 时应用启动崩溃的问题。
- 默认界面只保留额度仪表，其他项目数据改为悬停显示。
- 展开时顶部额度仪表保持原位，详情在下方纵向出现，避免横向遮挡页面。
- 详情按“当前项目（本周）／全部项目（本周）／历史总消耗”分成三行。
- 默认仪表使用固定深色表面、环形百分比和横向进度条。
- 默认尺寸为参考设计的 82%，可在 65% 至 125% 之间调整并自动保存。
- 支持左、右、上、下屏幕边缘吸附，状态球保留实时剩余百分比和额度颜色。
- 当前项目改为跟随 Codex 活跃顶层任务，切换任务或项目后 5 秒内更新名称与项目 Token。
- 详情图标统一使用低对比度中性色。
- 额度是唯一强调色：高于 25% 为绿色，10% 至 25% 为琥珀色，10% 以下为红色。
- 使用真实透明圆角窗口和固定深色表面，在浅色、深色页面上都保持边界清晰。
- 采用整体等比缩放，环形、文字、进度条和详情行同步变化，不抢 Codex 焦点。
- 隐私检查可在没有 `.git` 的独立桌面文件夹内运行。

## 重新构建和测试

在“终端”中进入本文件夹，然后运行：

```bash
zsh ./scripts/test.sh
open "build/Codex Quota HUD.app"
```

测试包括隐私扫描、空项目状态回归、格式计算、悬停交互契约、焦点安全、Universal 2 架构、签名及应用元数据。

## 数据与隐私

程序在本机读取 Codex 的额度和 Token 状态，不包含自身的网络上传、分析 SDK 或遥测。它只保存浮窗位置、缩放比例、边缘吸附状态和最近一次可用的历史 Token 总量。
