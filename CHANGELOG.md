# Changelog

## 1.2.1

- 修复 rollout 中 Token 事件的 `info` 为 `null` 时应用启动崩溃的问题。
- 增加对应回归测试，遇到不完整 Token 事件时安全跳过并继续统计。
- 使用 MIT License，允许自由使用、修改和分发。
- 增加中英文风险、安全、版权与第三方商标免责声明。

## 1.2.0

- 当前项目改为读取 Codex 顶部可见的任务名称，并与本机任务数据库匹配，切换任务后最多 5 秒更新。
- 首次运行可请求只读辅助功能权限；无权限时保留最近活跃任务与项目状态兜底。
- 补充辅助功能读取范围和隐私边界说明。

## 1.1.0

- 将菜单栏入口改为仅在 Codex 前台显示的磨砂浮窗。
- 显示额度剩余、当前项目本周消耗、全项目本周消耗、占比和历史总消耗。
- 当前项目跟随 Codex 的 `selected-project` 自动切换。
- 支持拖动位置、调整长度和字体自适应，并保存窗口几何信息。
- 主体点击穿透，切换到其他应用时隐藏，不请求焦点。
- 增加远端额度失败时的缓存与本地累计兜底。
- 发布包升级为同时支持 Apple 芯片和 Intel Mac 的 Universal 2。
- 增加独立隐私说明、双语文档、仓库隐私检查和 SHA-256 校验文件。
- 为 Codex `app-server` 响应增加超时保护。
